# Zynk Driver App

Delivery partner mobile application for Zynk Quick Commerce platform.

## Features
- Accept/reject delivery orders
- Real-time navigation to customer location
- Order status updates
- Earnings dashboard
- Availability toggle (online/offline)

## Tech Stack
- React Native / Expo
- Supabase (Backend & Auth)
- Google Maps API
- React Navigation

## Setup
```bash
npm install
npm start
```
